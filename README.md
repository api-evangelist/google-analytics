# Google Analytics API (google-analytics)
Google Analytics provides data and insights about website and app usage, enabling businesses to understand their audience and optimize their digital properties.

**URL:** [Visit APIs.json URL](https://analytics.google.com)

## APIs

### Google Analytics Data API (GA4)
The Google Analytics Data API provides programmatic access to Google Analytics 4 (GA4) report data.

**Human URL:** [https://developers.google.com/analytics/devguides/reporting/data/v1](https://developers.google.com/analytics/devguides/reporting/data/v1)


#### Tags:

 - Analytics, Reporting, GA4, Data API

#### Properties

- [Documentation](https://developers.google.com/analytics/devguides/reporting/data/v1)
- [OpenAPI](https://analyticsdata.googleapis.com/$discovery/rest?version=v1beta)
- [Authentication](https://developers.google.com/analytics/devguides/reporting/data/v1/quickstart-client-libraries)
- [Pricing](https://developers.google.com/analytics/devguides/reporting/data/v1/quotas)

### Google Analytics Admin API
The Analytics Admin API allows programmatic access to configuration data for Google Analytics 4 properties.

**Human URL:** [https://developers.google.com/analytics/devguides/config/admin/v1](https://developers.google.com/analytics/devguides/config/admin/v1)


#### Tags:

 - Admin, Configuration, Management, GA4

#### Properties

- [Documentation](https://developers.google.com/analytics/devguides/config/admin/v1)
- [OpenAPI](https://analyticsadmin.googleapis.com/$discovery/rest?version=v1beta)
- [Reference](https://developers.google.com/analytics/devguides/config/admin/v1/rest)
- [Quickstart](https://developers.google.com/analytics/devguides/config/admin/v1/quickstart-client-libraries)

### Google Analytics Reporting API v4 (Universal Analytics)
The Analytics Reporting API v4 provides programmatic access to Universal Analytics report data (note Universal Analytics sunset July 1, 2023).

**Human URL:** [https://developers.google.com/analytics/devguides/reporting/core/v4](https://developers.google.com/analytics/devguides/reporting/core/v4)


#### Tags:

 - Analytics, Reporting, Universal Analytics, Legacy, Deprecated

#### Properties

- [Documentation](https://developers.google.com/analytics/devguides/reporting/core/v4)
- [OpenAPI](https://analyticsreporting.googleapis.com/$discovery/rest?version=v4)
- [Migration Guide](https://developers.google.com/analytics/devguides/migration/api/reporting-ua-to-ga4)
- [Deprecation Notice](https://support.google.com/analytics/answer/11583528)

### Google Analytics Management API v3
The Analytics Management API allows access to configuration data for Universal Analytics accounts, properties, and views.

**Human URL:** [https://developers.google.com/analytics/devguides/config/mgmt/v3](https://developers.google.com/analytics/devguides/config/mgmt/v3)


#### Tags:

 - Management, Configuration, Universal Analytics, Legacy

#### Properties

- [Documentation](https://developers.google.com/analytics/devguides/config/mgmt/v3)
- [Reference](https://developers.google.com/analytics/devguides/config/mgmt/v3/mgmtReference)
- [Deprecation Notice](https://support.google.com/analytics/answer/11583528)
